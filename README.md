# 💳 UPI Transactions Analysis Dashboard  

**Digital Payments & Financial Analytics (Power BI)**  

This project presents an interactive **Power BI dashboard** analyzing UPI transaction data across multiple dimensions such as **cities, banks, transaction types, payment methods, and user demographics**.

The dashboard enables stakeholders to understand **transaction patterns, financial trends, and user behavior** in digital payments.

---

## 📌 Project Overview  

With the rapid growth of UPI-based transactions, analyzing large-scale financial data has become critical.  

This dashboard provides a **centralized analytical view** of transaction activity, helping identify trends across **time, geography, and user segments**.

---

## 🧰 Tech Stack  

- **Power BI Desktop** – Dashboard development & visualization  
- **DAX (Data Analysis Expressions)** – KPI calculations  
- **Power Query** – Data transformation  
- **CSV Dataset** – Source data  

---

## ❗ Business Problem  

Organizations face challenges in:  

- Tracking **monthly financial trends**  
- Understanding **city-wise transaction distribution**  
- Analyzing **user demographics and behavior**  
- Monitoring **balances and transaction flow**  

---

## 🎯 Objective  

- Analyze transaction patterns across multiple dimensions  
- Track **monthly transaction amounts and balances**  
- Identify **high-performing cities and segments**  
- Enable **data-driven financial insights**  

---

## 📊 Dashboard Features  

### 🔹 Monthly Balance Trend  
- Tracks **account balance variation across 12 months (2024)**  
- Helps identify financial stability and fluctuations  

### 🔹 Transaction Amount Analysis  
- Compares **monthly transaction volumes**  
- Detects peak activity periods  

### 🔹 City-wise Performance  
- Compares transaction **amount and balance across major cities**  
- Highlights regional contribution  

### 🔹 Interactive Filters  
- Bank (Sender & Receiver)  
- City  
- Device Type  
- Gender  
- Age Group  
- Merchant Name  
- Payment Method  
- Purpose  
- Transaction Type  

### 🔹 Bookmark Navigation  
- Implemented **bookmarks for seamless navigation** between views  
- Enhances user experience without drill-down
